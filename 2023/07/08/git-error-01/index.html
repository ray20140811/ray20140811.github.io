git - remote error refusing to update checked out branch refs/heads/main

2023-07-08

首先在server上新建一個目錄 instant_nodejs_starter ,並用 git init 命令創建了一個新的repository

在local端有 git clone 複製下來,增加了新的內容,做了 git add 和 git commit,但最後 git push 時,出現下面的錯誤

我以為是沒執行 git pull 導致,所以執行了一下 git pull,但還是出錯

看不懂這個錯誤是什麽意思,所以又返回第一個錯誤訊息仔細看了一下.從底下這句確認了錯誤的問題

突然想起,是我在創建新的repository時,忘了帶參數 –bare.後來重新到server,刪除原來的資料夾,
重新創建一個新的,這次指令改成

然後重新在local端重新 git push,這次就成功了.

所以要記得,創建新的repository記得要帶上 --bare
