Emacs - Learn Org Mode

2023-09-08

以前曾看過Org Mode,覺得很神奇,但因為要記憶很多命令,老是記不住,過一段時間就全忘了.
這次有了Blog,記錄一下學習的過程.

Org Mode基本使用

Command Key Description
org-cycle
org-shifttab S-
org-metaup M-
org-metadown M-
org-metaleft M-
org-metaright M-

Org Organizer

Command Key Description
org-todo C-c C-t You can customize the keywords via the variable ‘org-todo-keywords’
org-schedule C-c C-s
org-deadline C-c C-d
org-agenda
org-agenda-file-to-front C-c [ Once you have some TODO items planned in an Org file, you can add that
file to the list of “agenda files”. The list of agenda files is stored
in the variable ‘org-agenda-files’

Org Authoring

Command Key Description
org-export C-c C-e

To export several files at once to a specific directory, either
locally or over the network, you must define a list of projects through
the variable ‘org-publish-project-alist’.
