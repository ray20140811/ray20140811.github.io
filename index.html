GNU Emacs Reference Card (for version 28)

GNU Emacs Reference Card

Key Binding Notation
C-xCtrl + X
M-xAlt + X
S-xShift + X
S-M-xCtrl + Alt + X
Leaving Emacs
C-ziconify Emacs (or suspend it in terminal)
C-x C-cexit Emacs permanently
Error Recovery
C-gabort partially typed or executing command
C-x u, C-_ or C-/undo an unwanted change
C-lredraw garbaged screen
M-x recover-sessionrecover files lost by a system crash
M-x revert-bufferrestore a buffer to its original contents
Case Change
M-uuppercase word
M-llowercase word
M-ccapitalize word
C-x C-uuppercase region
C-x C-llowercase region
Keyboard Macros
C-x ( start defining a keyboard macro
C-x ) end keyboard macro definition
C-x e execute last-defined keyboard macro
C-u C-x ( append to last keyboard macro
M-x name-last-kbd-macro name last keyboard macro
M-x insert-kbd-macro insert Lisp definition in buffer
